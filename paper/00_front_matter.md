## Аннотация

> Объём аннотации не превышает 1500 знаков с пробелами (Прил. 1 Положения). Содержит цели и задачи работы, полученные результаты, рекомендации. Перед сдачей ВКР пересматривается с актуальными цифрами после стабилизации Главы 6.

В работе исследована применимость методов машинного обучения к обратной задаче мультипольного анализа: восстановлению коэффициентов разложения электромагнитного поля по векторным сферическим гармоникам по мощностной диаграмме направленности. Задача является некорректной по Адамару: множество прообразов содержит более одного элемента из-за глобальной фазовой и отражённо-сопряжённой неоднозначностей.

Цель работы — построить машинно-обученный регуляризатор, приближающий один физически разумный прообраз из множества решений, и собрать воспроизводимый бенчмарк. Задачи: формализация обратной задачи; разработка пайплайна обучающих данных (синтетика и реальные данные с аугментацией); проектирование архитектур моделей и функций потерь; выбор метрик качества; экспериментальная оценка вклада каждого рычага.

Полученные результаты: разработан пайплайн генерации обучающей выборки; предложена архитектура многослойного перцептрона с многоголовой структурой по принципу Матрёшки и дифференцируемым декодером векторных сферических гармоник; собран набор функций потерь, в том числе ранжирующая потеря на поле, существенно ускоряющая сходимость на реальных данных; разработан программный фреймворк mpinv с открытыми датасетами и конфигурациями.

Рекомендации: использовать ранжирующий компонент потерь для устойчивой сходимости; предпочитать концентрирующее по низким порядкам распределение коэффициентов при генерации синтетики; раздельно отчитываться по валидационной и отложенной подвыборкам.

## Обозначения и сокращения

### Сокращения

| Сокращение | Расшифровка |
|---|---|
| ВКР | выпускная квалификационная работа |
| ГЭК | государственная экзаменационная комиссия |
| AdamW | оптимизатор Adam с весовым затуханием (Adam with decoupled weight decay) |
| FFT | быстрое преобразование Фурье (fast Fourier transform) |
| MLP | многослойный перцептрон (multi-layer perceptron) |
| MSE | среднеквадратичная ошибка (mean squared error) |
| NRMSE | нормированный корень среднеквадратичной ошибки (normalised root mean squared error) |
| PCA | метод главных компонент (principal component analysis) |
| ReLU, SiLU | стандартные функции активации (rectified linear unit, sigmoid-weighted linear unit) |
| SH | сферические гармоники (spherical harmonics) |
| SVD | сингулярное разложение (singular value decomposition) |
| TE / TM | поперечно-электрическая / поперечно-магнитная мода |
| VSH | векторные сферические гармоники (vector spherical harmonics) |

### Основные обозначения

| Символ | Значение |
|---|---|
| $L$ | порядок усечения мультипольного разложения |
| $K = L(L + 2)$ | число пар $(l, m)$ при усечении $L$ |
| $a^E_{lm}, a^M_{lm}$ | электрические и магнитные мультипольные коэффициенты |
| $\boldsymbol\Psi^E_{lm}, \boldsymbol\Psi^M_{lm}$ | электрические и магнитные векторные сферические гармоники |
| $E_\theta, E_\varphi$ | комплексные тангенциальные компоненты электрического поля |
| $E_{UT}$ | комплексное поле тестируемого излучателя (Under Test) |
| $P(\theta, \varphi) = |E_\theta|^2 + |E_\varphi|^2$ | мощностная диаграмма направленности |
| $\mathcal S$ | оператор синтеза комплексного поля по коэффициентам |
| $\mathcal A = |\cdot|^2 \circ \mathcal S$ | прямой оператор «коэффициенты → мощность» |
| $\mathbf B$ | матрица аналитического разложения (см. § 1.3) |
| $\mu(\theta) = \sin\theta\,\Delta\theta\,\Delta\varphi$ | весовая функция площади на сфере |
| $\|\cdot\|_w$ | $L^2$-норма с весом $\mu(\theta)$ |
| $\hat a$ | предсказанный машинно-обученной моделью вектор коэффициентов |
| $R^2$ | коэффициент детерминации |

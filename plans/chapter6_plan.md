## Примечания по написанию

1. **Язык**: Только русский, без англицизмов
2. **Названия экспериментов**: Использовать описательные названия из proposal.md, не S0-S5
3. **Placeholders**: Для всех результатов, которые ещё не посчитаны, использовать формат:
   ```
   [PLACEHOLDER: Результаты эксперимента Step 0 будут добавлены после завершения расчётов]
   ```
4. **Консервативные утверждения**: Не делать сильных заявлений без численного подтверждения. Не делать заявлений без фактического подтверждения из достоверных источников.
5. **Связь с предыдущими главами**: Ссылаться на определения из Глав 1-5
6. **Бюджет страниц**: 5-7 страниц согласно THESIS_PLAN.md
7. **Общие положения**: Основываться на `paper/JUDGE_VKR.md` при написании главы.
